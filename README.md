# turtle-say

Simple discord bot to generate a turtle saying things

# Usage

```
!say <message//second line> [-s maxsize] [-p interline-pad [left-pad [right-pad]]]
```

# Dependencies

Server:

- discord.py (pip)

Image generation:

- svgwrite (pip)
- numpy (pip)

- imagemagick (`convert` command)
